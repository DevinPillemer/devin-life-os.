# Floopify (Flipper Fire)

A personal dashboard and learning platform built with React, Vite, and Tailwind CSS.

## Features

- Personal Finance Dashboard
- Learning Module System
- Daily Habits Tracking
- Health Metrics
- Goals Tracker

## Live Demo

Visit: https://feedly-news-dashboard.vercel.app

## Development

```bash
npm install
npm run dev
```

Auto-deploys to Vercel on push to main branch.
